## ⚠️ **Important Notes:**

### **Rate Limits Apply Per Project:**
- Limits are per Google Cloud project, NOT per API key
- Creating multiple API keys doesn't increase limits
- Need more? Create separate Google Cloud projects

### **Daily Reset:**
- RPD resets at **midnight Pacific Time**
- Plan heavy usage accordingly
- Monitor your usage in Google AI Studio

### **Error Handling:**
- If you hit limits, you'll get HTTP 429 error
- App will show friendly error message
- Retry automatically after cooldown
